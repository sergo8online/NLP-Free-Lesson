# NLP Lesson Notebook

This repository contains a comprehensive Jupyter Notebook covering fundamental and intermediate concepts in Natural Language Processing (NLP), with both theory and practical implementations.

## Contents

### 1. Tokenization and Tokenizers
- Basic text preprocessing
- Different tokenization strategies

### 2. Text Normalization
- Lowercasing
- Removing punctuation
- Cleaning and preparing text data

### 3. Stemming and Lemmatization
- Core concepts
- Practical tasks and examples

### 4. Embeddings
- Introduction to vector representations of text
- Understanding how words are encoded numerically

### 5. Recurrent Neural Networks (RNNs)
- Basic RNN architecture
- Sequence modeling concepts

### 6. Sequence Modeling (End-to-End)
- Full pipeline implementation
- Training an RNN to predict the next token based on previous tokens

### 7. Bidirectional RNNs
- Concept and advantages over standard RNNs

### 8. LSTM
- Handling long-term dependencies
- Improved sequence learning

### 9. Text Classification with RNN
- Building and training an RNN-based classifier
- Applying NLP pipeline to a real task

## Tech Stack
- Python
- PyTorch
- Jupyter Notebook

## Purpose
This notebook is designed as a structured learning resource for NLP, combining theoretical explanations with hands-on implementations to build a solid foundation in sequence modeling.

## Usage
Open the notebook and follow the sections sequentially. Each part includes explanations and code examples for better understanding.
